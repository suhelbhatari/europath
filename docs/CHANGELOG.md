# Changelog

## [1.0.0] - June 2025

### Added
- 45 European countries with full immigration data
- 180+ PR pathways, 250+ citizenship pathways documented
- React web app with D3-geo political map of Europe
- Hash routing with deep-linkable pathway anchors
- Side-by-side comparison (up to 4 countries)
- Android app (Kotlin + Jetpack Compose + Material 3)
- Node.js REST API with Swagger docs
- Excel workbook with 12 sheets and live Country Search box
- GitHub Actions CI/CD workflows
- Full documentation suite

### Countries Covered
Germany, France, Portugal, Spain, Italy, Netherlands, Switzerland, Sweden, Norway,

Slovakia, Hungary, Romania, Bulgaria, Serbia, Montenegro, Albania, North Macedonia,
